# Skill Registry - profesionalesecfront

## Compact Rules

### General
- Use Conventional Commits.
- Maintain "Apple-inspired" premium aesthetic: clean layouts, ample whitespace, legible typography.
- Ensure mobile-first responsiveness (no overflow at 320px).
- Use `oklch` for colors as defined in `app/globals.css`.

### Next.js & React
- Use App Router patterns.
- Prefer Server Components where possible; use `'use client'` sparingly.
- Use Framer Motion for micro-interactions.

### Styling
- Tailwind CSS v4 usage.
- shadcn/ui components for base UI.
